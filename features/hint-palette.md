# Using the hint palette

The hint palette shows the user are the commands to use the app.

## Use the escape key to display the hint palette to the right 
* Given I am on the Matins main page
* When I type the Escape key
* Then the hint palette displays to the right

## Type colon to display the hint palette to the right
* Given I am on the Matins main page
* When I type ":" 
* Then the hint palette displays to the right

## Click "Palette" to display the hint palette to the right
* Given I am on the Matins main page
* When I click on the "Hints" option
* Then the hint palette displays to the right 

## Closing the hint palette with the Escape key
* Given the hint palette is open
* When I type the Escape key
* Then the hint palette closes

## Closing the hint palette with the Close option
* Given the hint palette is open
* When I click the Close option
* Then the hint palette closes
